# Task Completion Checklist

When completing a task, ensure:

1. **Tests pass**: `pytest tests/` (Python), `cargo test` (Rust)
2. **Linting clean**: `ruff check python/`, `cargo clippy` (Rust)
3. **Formatting**: `ruff format python/`, `cargo fmt` (Rust)
4. **Commit**: Descriptive commit message summarizing the change
